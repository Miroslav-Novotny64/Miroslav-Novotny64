## Hlavní projekty

### Expertov (Privátní repozitář)
Produkční full-stack platforma vyvíjená v rámci maturitní práce a již provozovaná jako SaaS řešení.
* **Stack:** Next.js 15 (App Router, Standalone), TypeScript, tRPC, TanStack Query.
* **Infrastruktura:** * Provoz na vlastním Linux VPS přes Docker Compose.
    * Automatizovaný deployment pomocí GitHub Actions (SSH/Docker build).
    * Zabezpečení síťového provozu přes Cloudflare Tunnels.
    * Caching a správa dat přes PostgreSQL 17, Drizzle ORM a Redis.

### [Flashscore API Service](https://github.com/Miroslav-Novotny64/FlashscoreScraping)
Vlastní modifikace scraperu transformovaná na JSON API. Slouží jako datový zdroj pro platformu Expertov.
* **Optimalizace:** Přepracování logiky z ukládání do souborů na přímé API rozhraní.
* **Výkon:** Zvýšení efektivity snížením počtu scrapovaných stránek (Místo scrapování N stránek (počet zápasů), stačí pouze 2).

### [Simple Impostor Game](https://github.com/Miroslav-Novotny64/simple-impostor-game)
Progresivní webová aplikace (PWA) postavená v Reactu. Projekt zaměřený na mobilní UX a offline funkcionalitu.
* **Motivace:** Vytvoření vlastní funkční alternativy k aplikacím, které obsahovali placené funkce, nebo hodně reklam.
